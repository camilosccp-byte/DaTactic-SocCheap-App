# DaTactic-SocCheap-App: Democratizing Data for Low-Budget Clubs

## 📌 Project Overview
Data science shouldn't be exclusive to elite clubs with multi-million dollar budgets. This repository contains a practical R application designed specifically for technical staffs operating under strict financial limitations. It bridges the gap by allowing any club member to extract professional tactical insights from free and accessible scouting platforms.

## 📊 Methodology & Technical Stack
- **Language:** R
- **Features:** Automated processing of open-source eventing and scouting datasets.
- **Target Audience:** Grassroots, youth academies, and lower-division technical staffs.

## 🚀 Key Insights for Football Clubs
1. **Budget Optimization:** Demonstrates how to run an analytical department with zero software licensing costs.
2. **Tactical Accessibility:** Empowers scouts and assistant coaches to generate performance profiles and eventing analysis effortlessly.
3. **Data Literacy:** Serves as an entry point for clubs transitioning into modern, data-driven football workflows.
